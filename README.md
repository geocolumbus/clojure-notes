# clojure-notes

Notes on the Clojure Programming Language

## Installation

Install [leiningen](https://leiningen.org/)
```
git clone https://github.com/geocolumbus/clojure-notes.git
cd clojure-notes
```

## Usage

Make sure you are in the ```clojure-notes``` folder.

```
lein run
```
